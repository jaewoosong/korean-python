# Korean Python
파이썬의 주요 함수 및 클래스를 한글 코딩으로 재구현하고 있습니다.

## 목적
한글 코딩으로도 깊이있는 개발을 할 수 있음을 증명하는 것이 목적입니다.

## 작업 원칙
1. 직역보다는 의미를 살리는 번역을 지향합니다.
2. 한국어 단어 사용을 원칙으로 하되 유용한 경우 외국어를 소리나는대로 옮깁니다.
3. 가능한 한 개발자의 타자 수를 줄여주는 쪽으로 번역합니다.
